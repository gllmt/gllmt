## Featured work

**[open-poker-planning](https://github.com/gllmt/open-poker-planning)**: real-time planning poker for agile teams. No login, token-based rooms, live sync across everyone in the room. Built with the Next.js App Router and Convex.

**[FrameWatch](https://github.com/gllmt/FrameWatch)**: browser extension that overlays a minimal in-page HUD with live video playback quality metrics. WXT, React, TypeScript, Shadow DOM isolation, EN/FR.

**[macos-perf-audit](https://github.com/gllmt/macos-perf-audit)**: CLI that audits Apple Silicon Mac performance (CPU, memory, disk I/O, GPU power) and writes a structured JSON report with anomalies and optimization hints.
